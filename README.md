# mackerel-plugin-macos-powermetrics

Mackerel metric plugin for MacOS system metrics.

## Usage

must be invoked as superuser.

```
Usage:
  ./mackerel-plugin-macos-powermetrics

Options:
  -metric-key-prefix string
        Metric key prefix (default "powermetrics")
  -tempfile string
        Temp file name
  -h, --help
        show help
```
